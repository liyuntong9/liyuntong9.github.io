Chrome

Ctrl+N打开新窗口

Ctrl+T打开新标签页

Alt+F4/Ctrl+Shift+W关闭当前窗口

Ctrl+W关闭当前标签页

F5刷新页面

F6/Ctrl+L定位到地址栏

F12打开开发者工具

Shift+F6跳出地址栏

Ctrl+K默认搜索

Ctrl+D加入收藏夹

Ctrl+Tab切换标签页

按住Ctrl键点击超链接，在新标签页打开

按住Shift键点击超链接，在新窗口打开

chrome://settings/

chrome://extensions/

Vimium

j,d向下翻页

k,u向上翻页

gg,G定位到页面的顶部和底部

f在新标签页打开链接

yy复制当前地址

yt复制当前标签页到新标签页

gi定位第一个输入框

J,K切换标签页

b搜索书签

w在窗口打开标签页

搜索引擎

baidu

github

bilibili

scholar

wikipedia

Windows

Ctrl+Shift+N新建文件夹

win+I打开设置

win+E打开资源管理器

F2 重命名文件夹
